# Why Antidote?

In short, it's the fastest and simplest manager, while automating the boring parts.

## Why not oh-my-zsh

I originally used oh-my-zsh, but it was too slow.

## Why not manually manage plugins

Sure that's just git clone, some fpaths, etc, but it's already work that we might as well use a tool for. Plus, antidote supports oh-my-zsh plugins out of the box.

## What about Zinit?

Never tried it, and also seems to have some funky political history that I would like to avoid.
