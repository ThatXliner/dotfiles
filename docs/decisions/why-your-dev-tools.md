# Why your dev tools?

The dev tooling in question here will be:

1. Terminal
2. Browser
3. Editor


## Terminal

I've tried Terminal.app, iTerm2, and Ghostty. I've liked iTerm2, but when I found out about iTerm2's security vulnerability that they refuse to patch, I switched to Ghostty. I haven't tried Alacritty or Vercel's terminal yet, and I don't plan to.

Kitty is Linux-only (I think?), so I don't use it.

I actually use foot on Nix though

## Browser

My journey was:

Safari -> Chrome -> Safari -> Firefox -> Safari -> Arc -> Zen

Zen's completely open source so that's an automatic win. It has some memory leak issues occasionally though

## Editor

My journey was:

Sublime Text -> Atom -> VS Code -> Zed

Long live Atom (Zed is made by the same people who created Atom originally)
